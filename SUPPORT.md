# Support

## Use GitHub Issues For

- bug reports
- feature requests
- documentation corrections
- CI/build regressions

## Use Security Reporting For

- vulnerabilities and exploit paths
- supply-chain compromise reports

Follow `SECURITY.md` for private disclosure.

## Discussions

If GitHub Discussions is enabled, use it for design exploration and non-blocking
implementation questions. If Discussions is not enabled, use Issues with prefix
`DISCUSS:` for design threads.
